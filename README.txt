Αρχικά, υλοποιήσαμε την εφαρμογή σε NetBeans και την βάση δεδομενων σε MySQL μέσω Workbench, καθώς και Glassfish ως server.
Πάνω στην html χτίσαμε τις έξτρα λειτουργίες με javascript και java και για την ασφάλεια των σελιδών χρησιμοποίησαμε servlets.

Πιο αναλυτικά για την βάση δεδομένων έχουμε τα εξής tables:
	Users: Πίνακας χρηστών
	Items: Πίνακας των αντικειμένων για δημοπρασία
	Photos: Πίνακας για την αποθήκευση των φωτογραφιών κάθε δημοπρασίας
	Categories: Πίνακας για την αποθήκευση των κατηγοριών κάθε δημοπρασίας
	Category_labels: Πίνακας για την αποθήκευση των κατηγοριών
	Won: Πίνακας κερδισμένων δημοπρασιών
	Bids: Πίνακας πονταρισμάτων
	Bidder: Πίνακας στοιχείων bidder
	Messages: Πίνακας αποθήκευσης μηνυμάτων
